# Madrasa-e-Riyaziyat 2026

This repository contains the lesson planning documents and curriculum design for **Madrasa-e-Riyaziyat**, a one-week mathematical summer program organized by the Math Club.

## Overview

Madrasa-e-Riyaziyat is designed to introduce students to mathematical thinking beyond the standard school curriculum. The program emphasizes problem solving, logical reasoning, visualization, and mathematical exploration through interactive activities, puzzles, and classical mathematical problems.

## Repository Contents

```text
.
├── plan.tex
└── README.md
```

* `plan.tex` contains the LaTeX source for the lesson plan and curriculum outline.

## Compilation

To compile the lesson plan:

```bash
pdflatex plan.tex
```

or

```bash
latexmk -pdf plan.tex
```

## Objectives

The program aims to:

* Develop mathematical intuition.
* Encourage creative problem solving.
* Expose students to modern mathematical ideas.
* Demonstrate real-world applications of mathematics.
* Foster collaborative learning and discussion.
